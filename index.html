十大排序算法实现与总结

发表于 2019-02-23 |

　　　　十大排序算法实现与总结

冒泡排序：

def bubble_sort (lists):
　　length = len(lists)
　　for index in range(length):
   　　　for i in range (1, length-index):
       　　if lists[i-1] > lists[i]:
            　lists[i], lists[i-1] = lists[i-1],lists[i]
　　return lists

　　持续每次对越来越少的元素重复比较相邻的，直到没有任何一对数字需要比较,这就会有最好的时间复杂度O(n)

选择排序：

def select_sort (lists):
　　length = len(lists)
　　for index in range(length):
    　　for i in range(index+1, length):
        　　if lists[index] > lists[i]:
            　　lists[index], lists[i] = lists[i], lists[index]
　　return lists

插入排序:

def insertion_sort(lists):  
    length = len(lists)
    for index in range(1, length):
        key = lists[index]
        for i in range(index-1,-1,-1):
            if lists[i] > key:
               lists[i+1],lists[i] = lists[i],lists[i+1]
               key = lists[i]
            else:   #最好时间复杂度O(n)
                break
    return lists

　　以上三种空间复杂度为O(1),平均时间复杂度为O(n2),最坏O(n2),最好插入和冒泡为O(n),选择为O(n2)

希尔排序：

def shell_sort(lists): #插入排序改进版
　　length = len(lists)
　　dist = length // 2
　　while dist > 0:
    　　for i in range(dist,length):
        　　item = i
       　　 while (item-dist) >= 0:
            　　if lists[item] < lists[item-dist]:

   　　 dist //= 2

平均时间复杂度O(n1.3),最坏O(n2),最好O(n),空间复杂度O(1).

归并排序:

def merge_sort(lists):
    if len(lists) <= 1 :
        return lists
    mid = len(lists) // 2
    left = merge_sort(lists[:mid])
    right = merge_sort(lists[mid:])
    return merge(left,right)
def merge(left,right):

    result = []
    i = 0
    j = 0

    while len(left) > i and len(right) > j:
        if left[i] < right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1
    result += left[i:]
    result += right[j:]
    return result

　　平均时间复杂度,最坏时间复杂度,最好时间复杂度均为O(nlog(n)),空间复杂度为O(n)

快速排序：

def quick_sort(lists,low,high):
    if low < high:
        key_index = partition(lists,low,high)
        quick_sort(lists,low,key_index)
        quick_sort(lists,key_index + 1,high)

def partition(lists,low,high):
    key = lists[low]
    while low < high:
        while low < high and lists[high] >= key:
            high -= 1
        if low < high :
            lists[low] = lists[high]

        while low < high and lists[low] < key:
            low += 1
        if low < high :
            lists[high] = lists[low]
        lists[low] = key
    return low

　　平均时间复杂度O(nlogn),最坏时间复杂度O(n2),最好时间复杂度O(nlogn),空间复杂度O(logn)　　　　　　　　　　　　

　　快速排序是对列表本身做调整,故无返回值.对倒序排列时间复杂度最坏,退化到冒泡排序.故越无序用排序越好.

堆排序 :

def heap_ajust (L, start, end):
    while start >= 0:
        root = start

        while True:
            child = 2 * root + 1
            if child > end :
                break

            # 两个孩子找出最大的那个
            if child + 1 <= end and L[child] < L[child + 1]:
                child += 1

            if L[root] < L[child]:
                L[root], L[child] = L[child], L[root]
                root = child
            else:
                break

        start -= 1
    return L 

def heap_sort (L):
     L_length = len(L)
     start = L_length // 2 - 1
     while L_length != 0:
         L = heap_ajust(L, start, L_length - 1)
         L[L_length - 1], L[0] = L[0], L[L_length - 1]
         L_length -= 1
     return L

堆排序最坏最好平均时间复杂度均为O(nlog(n)),空间复杂度为O(1)

计数排序 :

def counting_sort(L):
    # 找最大值
    max = L[0]
    for i in range(1,len(L)):
        if L[i] > max:
            max = L[i]

    item = [0 for i  in range(max+1)]
    for i in L:
        item[i] += 1

    result = []
    for j in range(len(item)):
        if item[j] != 0:
            result.append(j)
            item[j] -= 1
    return result

　　 计数排序是将待排序数值范围做新列表底i,而将每个值的个数做item[i],重新输出到result.可找待排序最小值以缩短时间,适合排序数值密集型的,并且不能有小数和负数.

　　最好最坏以及平均时间复杂度均为O(n+k),空间复杂度也是O(n+k)

桶排序 : 

def bucket_sort(L):
   _max = max(L)
   # 定义桶的数量及大小
   item = [0 for i in range(_max // 10 + 1)]
   for i in range(len(item)):
       item[i] = []
   # 把相应数值放入桶中
   for i in L:
       item[i // 10].append(i)

   # 每个桶排序
   for i in range(len(item)):
       insertion_sort(item[i])
   #输出结果
   result = []
   for i in range (len(item)):
       for j in range(len(item[i])):
           result.append(item[i][j])
   return result

　　桶排序最好时间复杂度是O(n),最坏时间复杂度是O(n2),平均时间复杂度O(n+k),空间复杂度为O(n+k).桶排序最好情况下使用线性时间O(n)，桶排序的时间复杂度，取决与对各个桶之间数据进行排序的时间复杂度，因为其它部分的时间复杂度都为O(n)。很显然，桶划分的越小，各个桶之间的数据越少，排序所用的时间也会越少。但相应的空间消耗就会增大。

　　桶排序是计数排序的升级版。它利用了函数的映射关系，高效与否的关键就在于这个映射函数的确定.　桶排序的工作的原理：假设输入数据服从均匀分布，将数据分到有限数量的桶里，每个桶再分别排序（有可能再使用别的排序算法或是以递归方式继续使用桶排序进行排）.

基数排序：

def radix_sort(L, digit):
    for i in range(digit):
        s = [[] for j in range(10) ] #做桶
        for item in L:
            s[item // (10 ** i) %10].append(item)
        L = [m for k in s for m in k]
    return L

　　最好最坏以及平均时间复杂度为O(k*n), 空间复杂度为O(n+k), 其中 k << n.

　　基数排序的性能比桶排序要略差,因其每次放入桶中后还要拿出来,故时间复杂度略高.

Cassandra 原理

发表于 2019-01-23 |

　　　　Cassandra原理　

　　　最近学cassandra，通过问题方式学习cassandra基本原理。

  1. 　Cassandra 读取过程 

    keyspace　->　columnFamily　->　key　->　columnName　->　valume　-> timestemp

  2. 　Cassandra 单点失效
    　 Cassandra采用p2p模式，无中心节点，即无管理节点，不存在单点失效问题．

  3. 　 Cassandra中的key
    　　key相当于DBMS的主键，区分不同行．column是列名，key相当于特殊的列名．

  4. 　 Cassandra中的分区键　
    　　分区键负责数据分发，相当于DBMS主键包括两个以上的列．

  5. 　 列排序
    　　cassandra中一行可以包括很多列，且不需要对应，当一行有很多列时查找变得困难，故可用一行中的列排序，提高查找效率．列排序只按一行中列名进行排序，不排序列值．

  6. 　 cassandra 中的CAP
    一致性（Ｃ）：　
    　分为两个，strong consistent 

    Ｗ　+　Ｒ　＞　Ｎ

    　 week consistent 

    Ｗ　+　Ｒ　＜＝　Ｎ

    　N — 数据复制的份数
    　W — 更新数据时需要保证写完成的节点数
    　R — 读取数据的时候需要读取的节点数
    cassandra 根据　timestamp　判断最新值　　
    可用性（Ａ）: 所有节点数据，客户端总是可以读取，可用性较高.
    分区容错性（Ｐ）

  7. 　cassandra读写问题　　　　　　　
    　　提高写速度:　写时不读（故cassandra　写＞读）
    　　提高读速度:　修改只写不改，后面会通过压紧方式清理．　　

　8.　 p2p与master/serve 读写区别
　　　　p2p : 节点全可读可写，对cassandra来说写时不读．
　　　　master/serve: 节点全可读，写时只有从master节点开始．
