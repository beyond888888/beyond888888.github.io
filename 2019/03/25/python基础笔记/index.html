    • 基础类包括: 类(class)，类变量(name = ‘’)，构造方法，方法(def)，局部变量，类的私有属性、私有方法，实例化对象，运算符重载．



      //　构造方法：((def init()))有构造方法的实例化时可以有参数.

      //　self： self代表的是类的实例，而非类．self不是关键字，可以用其他符号代替，self.class 代表类

      // 类的私有属性、私有方法：以两个下划线开头，不能在类的外部或直接访问．

      // 运算符的重载：对类的专有方法（如：add,sub等）进行重载,重载这两个方法就可以在普通的对象上添加＋－运算符操作。

    • 父与子类关系: 继承，方法重写，多继承．



      //　多继承:(class child(base1,base2) 圆括号中父类的顺序，若是父类中有相同的方法名，而在子类使用时未指定，python从左至右搜索 即方法在子类中未找到时，从左到右查找父类中是否包含方法。

      //　方法重写: 调用父类方法,如果满足需求,不用在子类中写,如果不满足,相同名字方法在子类中重写,也可以在子类中添加其他父类没有的方法,重写后若想调用原父类方法可用超类super(方法名)．
