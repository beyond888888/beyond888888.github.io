  • python多线程

    • 多线程与多进程区别

      　　线程就是把一个进程分为很多片，每一片都可以是一个独立的流程。这已经明显不同于多进程了，进程是一个拷贝的流程，而线程只是把一条河流截成很多条小溪。它没有拷贝这些额外的开销，但是仅仅是现存的一条河流，就被多线程技术几乎无开销地转成很多条小流程，它的伟大就在于它少之又少的系统开销。（当然它也引发了重入性等种种问题）．

      　　但在Linux系统中,线程和进程不特别区分,线程是一种特殊的进程.进程包括两种虚拟机制，虚拟内存和虚拟处理器．线程之间可以共享虚拟内存,但每个线程有独立的虚拟处理器.所以linux系统开启线程也是用fock()，只是开启的时候指定共享的资源即可．
    • 常用的进程定义：进程描述符，僵尸进程，孤儿进程

      //　进程描述符：包含一个具体进程所有信息．

      //　僵尸进程：一个进程使用fork创建子进程，如果子进程退出，而父进程并没有调用wait或waitpid获取子进程的状态信息，那么子进程的进程描述符仍然保存在系统中。这种进程称之为僵尸进程。

      //　孤儿进程：一个父进程退出，而它的一个或多个子进程还在运行，那么那些子进程将成为孤儿进程。孤儿进程将被init进程(进程号为1)所收养，并由init进程对它们完成状态收集工作。
    • 常用的线程定义：父线程，子线程，守护线程．

      //　守护线程：当主线程结束时，结束其余的子线程（守护线程）自动关闭，就免去了还要继续关闭子线程的麻烦。否则，父线程结束，有可能子线程还在运行．thread.setDaemon(true)必须在thread.start()之前设置．
    • 常用线程函数和类：_thread 和 threading 是常用的对线程支持的标准库,threading包括前者所有方法,故直接学习threading即可.threading常用类和函数如下:
      • threading.Thread()
        用于处理线程，包括join([]time)，run(),start()，isAlive()，getName()，setName()

        //　join（[time]）的作用是，在子线程完成运行之前，这个子线程的父线程将一直被阻塞。这阻塞调用线程直至线程的join() 方法被调用中止(正常退出或者抛出未处理的异常-或者是可选的超时发生)。

        //　run()表示线程活动的所有方法

        //　start()启动线程,启动后自动调用了run()
      • threading.Condition()
        包括acquire()，release(),wait(),notify(),notifyAll().

        //　acquire()，release()用于获得和释放锁

        //　wait():释放底层锁，然后阻塞，直到它被另一个线程中的相同条件变量唤醒notify()或notifyAll()调用，或者直到发生可选超时。一旦被唤醒或超时，它就会重新获得锁定并返回.

        //　notify()唤醒一个等待条件变量的线程，如果有的话正在等待; 如果没有线程在等待，那么这是一个无操作。放在release()之前acquire()之后，即此线程必须获得锁定．notifyAll()是唤醒所有．

        //　wait() 方法notify()和notifyAll()方法。只有在调用线程获得锁定时才调用这三个.

  • Python日期和时间

    • time模块

      • 程序耗时 －－time.time()返回当前浮点时间，可以用两个函数衡量程序
      • 时间格式转换
        1. 返回时间元组：time.localtime(time.time())
        2. 返回时间：Thu Apr 7 10:36:20 2016 time.asctime(time.localtime()) 或　time.ctime()
        3. 时间格式化：time.strftime(“%Y-%m-%d %H:%M:%S”, time.localtime())　time.strftime(“%a %b %d %H:%M:%S %Y”, time.localtime())

          －－这里格式化[%y （19）%Y（2019）]年，［%m］月，[%d]日，[%H（0-23）%I（01-12)]时，［%M］分，[%S]秒，[%a （mon）%A（monday）]星期，[%b(feb),%B(feberary)]月,[%j] 年内的第几天．
      • 推迟调用线程的运行　 time.sleep(secs)

    • calendar模块（模块的函数都是日历相关的）

      • 某年某月日历：calendar.month(year,month,w=2,l=1)
      • 关于闰年：calendar.isleap(year)；calendar.leapdays(y1,y2)

  • python正则表达式(re模块)

    • re.match()
      • re.match 尝试从字符串的起始位置匹配一个模式，如果不是起始位置匹配成功的话,返回None,只从开始位置匹配,匹配一次.
    • re.search()
      • re.search 扫描整个字符串并返回第一个成功的匹配。
    • re.findall()
      • match 和 search 是匹配一次, findall 匹配所有
    • re.finditer
      • 和 findall 类似，在字符串中找到正则表达式所匹配的所有子串，并把它们作为一个迭代器返回。
    • re.compile()
      • compile 函数用于编译正则表达式，生成一个正则表达式（ Pattern ）对象，供 match() 和 search() 这两个函数使用
    • re.sub()
      • re.sub用于替换字符串中的匹配项
    • re.split()
      • split 方法按照能够匹配的子串将字符串分割后返回列表
    • 上面函数返回对象:
      • group() 返回被 RE 匹配的字符串。
      • span() 返回一个元组包含匹配 (开始,结束) 的位置
      • start() 返回匹配开始的位置
      • end() 返回匹配结束的位置
    • 最后需掌握常用flag,正则表达式模式,特殊字符.
